Version Xcode : 14.1

Structure de l'application :
Une vue principale qui affiche une liste de villes 
Une vue pour ajouter une ville à la liste de villes quand on appuie sur le bouton + de la vue principale
Une vue pour afficher les données meteorlogiques de la ville quand on clique sur une ville de la liste

Une classe pour representer une ville et stocker ses données
Une classe pour la liste des villes
Une classe pour la connexion avec les API
